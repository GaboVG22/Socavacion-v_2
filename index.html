Secciones del tramo

Puede usar geometría dibujada o ingresar ancho/tirante manual si aún no tiene levantamiento.

La columna “Curva” permite marcar secciones dentro de meandro o curva. El factor de curva se aplica como ajuste preliminar sobre la socavación calculada; se recomienda revisar con distribución lateral de velocidades o modelo 2D cuando sea crítico.

Dibujar o digitalizar sección transversal

Dibuje con lápiz o cargue una imagen y marque la línea del terreno. Luego guarde la geometría para la sección activa.

Clic o arrastre para agregar puntos. Doble clic elimina el último punto. Use una imagen de fondo si tiene perfiles exportados desde HEC-RAS, CAD, PDF o planillas.

Metodología incorporada en esta versión

Hidráulica por sección

Para geometrías dibujadas se calcula área mojada, perímetro mojado, radio hidráulico, ancho superficial, tirante medio, velocidad media y Froude. Si existe Q y pendiente, la profundidad normal se obtiene por iteración usando Manning.

Granulometría

La curva se interpola en escala logarítmica para estimar D50, D84, D90 y Dm. D90 alimenta la rugosidad de partícula en MPM; Dm alimenta Lischtvan-Levediev.

Transporte de sedimentos MPM

Se aplica Meyer-Peter-Müller con corrección de rugosidad tipo Strickler. Si la tensión efectiva no supera el umbral, el gasto sólido se informa como cero.

Socavación general Lischtvan-Levediev

Se calcula Hs para material granular homogéneo y la socavación como max(Hs − h, 0). Incluye β por período de retorno, z por Dm, μ de contracción y φ por mezcla agua-sedimento.

Tramos en curva

El marcado de curva no reemplaza un modelo 2D. En esta versión se usa como ajuste preliminar configurable. La recomendación técnica es discretizar la sección por franjas o aplicar modelación 2D para curvas críticas.

Alcance

Herramienta preliminar de revisión y comparación. No sustituye levantamiento topobatimétrico, calibración hidráulica, mediciones granulométricas ni validación profesional.
